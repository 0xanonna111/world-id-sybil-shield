# World ID Sybil Shield

In 2026, the proliferation of AI agents makes robust **Sybil Resistance** a core requirement for on-chain airdrops, governance voting, and quadratic funding protocols. This repository provides a professional-grade reference architecture for integrating **World ID** into your smart contracts and dApp backends.

The integration leverages Zero-Knowledge Proofs (ZKPs) generated by the World App (via the Orb biometric verification system). This allows users to prove they are unique humans without exposing their real-world identity, passport data, or personal details on public ledgers.

## Operational Lifecycle
1. **User Action:** The user generates a cryptographic proof of uniqueness on their mobile device via the World ID IDKit SDK.
2. **Relay Verification:** The generated ZK-proof, nullifier hash, and root parameters are packaged into an on-chain transaction.
3. **On-Chain Attestation:** The target smart contract queries the official WorldID router precompile/verifier to mathematically confirm the proof validity before dispensing rewards.

## Configuration Steps
1. Install project layout packages: `npm install`
2. Input your `app_id` and unique action parameters generated in the Worldcoin Developer Portal into `.env`.
3. Compile structural files using Hardhat: `npx hardhat compile`
